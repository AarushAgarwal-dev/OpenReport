# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-18

### Added
- Initial release of OpenReport Framework
- Dynamic 3-step report builder wizard
- Advanced filtering engine with type-aware operators
- Data visualization with Recharts (Bar, Line, Pie charts)
- Aggregations and grouping functionality
- Scheduling UI for daily, weekly, monthly reports
- CSV export functionality
- Enterprise dark theme UI
- Mock data generator for demo purposes
- React Context-based state management

### Features
- Support for multiple data sources
- Column selection and customization
- Sorting and filtering capabilities
- Real-time chart generation
- Report saving and management
- Responsive design
